To include amara videos on a page:

  • Add the following JavaScript, preferably early in the page: 
    	    <script type="text/javascript" src='http://s3.amazonaws.com/s3.www.universalsubtitles.org/release/public/embedder-iframe.js'>
	    </script>
  • Include these divs where you want to include the widgets, adding the video's URL as data-url: 
    	    <div class="amara-embed" style="height: 480px; width: 854px" data-url="http://www.youtube.com/watch?v=bNySL-ihAvo"></div>

    You can set the following options:

    • Hide the Amara logo by adding data-hide-logo="true" in the previous tag.
    • Hide the menu item to order subtitles by adding data-hide-order="true" in the previous tag.
    • Set the initial active subtitle language by adding data-initial-language="language" in the previous tag, where language is the language code, such as "en" for English.
    • Display the subtitles by default by adding data-show-subtitles-default="true" in the previous tag.
    • Display the transcript by default by adding data-show-transcript-default="true" in the previous tag.
